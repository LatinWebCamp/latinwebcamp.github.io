Challenge

Create a page with the following elements:

  1. At least 2 images, one of which is also a link.
  2. 2 styles using a different font, color, and size.
  3. 1 style using the CSS property to make a border.
  4. Change the h1 tag so that it is centered and a different color.
  5. Create a style that will indent the first line by 30 pixels.
  6. Create a style using one unique CSS property of your choosing.
