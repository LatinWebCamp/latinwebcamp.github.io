Challenge

Create a page about a famous athlete with the following elements:

  1. An image gallery (see puppies above for example) with each image also being a link.
  2. At least 3 paragraphs of text.
  3. 2 text styles using a different font, color, and size.
  4. Change the h1 tag so that it is centered and a different color, and has a solid blue border 5 pixels wide.
  5. Each paragraph should be indented by 20 pixels.
  6. The page should have a background image that is scaled to take up the entire width of the page.
