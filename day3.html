Challenge

Create a page about a famous athlete with the following elements:

  1. Change the h1 tag so that it is centered and a different color, and has a solid grey border 5 pixels wide.
  2. At least 3 paragraphs of text.
  3. Each paragraph should be a different font, color, and text size.
  4. Each paragraph should be indented by 20 pixels.
  5. An image gallery (see puppies above for example) with each image also being a link.

Note: you should to assign multiple classes to your paragraphs!
