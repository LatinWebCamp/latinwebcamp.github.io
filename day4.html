Assignment

Create a page with the following elements:

  1. 2 styles using a different font, color, and size. Use each style in at least one paragraph.
  2. At least 2 images and 2 links. The images should be sized so that they will fit on the page.
  3. Create a style that will add a border to an element that it is assigned to. Use this style in at least one paragraph.
  4. Change the h1 tag so that it is centered and a different color and font. Use the h1 tag to create a heading at the top of your page.
  5. Create a style that will indent the first line of each paragraph by 30 pixels.
  6. Use CSS to create a box with an image inside with a label. The image and text should be centered in the box. See: Box Example
  7. BONUS: Create three boxes as above, each next to each other, and sized to fit the image neatly inside the box.
  8. BONUS: Use CSS to put a border around the entire page.
  9. BONUS: Create a style using one unique CSS property of your choosing.
